# Project: Demo "Leer archivo .csv con react-csv-reader"

## Description

This project implements:

Package used: [React component that handles csv file input](https://github.com/nzambello/react-csv-reader)

### Install

```bash
cd tmpDemo_read_csv
npm install
```

### Compile

```bash
npm run build
```

### Execute

```bash
npm run start
```
